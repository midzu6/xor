# XOR Cipher CLI

Простая CLI утилита для шифрования и дешифрования текста с помощью XOR алгоритма. Зашифрованный текст кодируется в Base64 для удобной передачи.

## Как это работает

Каждый символ текста XOR-ится с символом ключа. Ключ повторяется циклично если он короче текста. Для расшифровки используется тот же ключ — двойной XOR возвращает исходный текст.

## Запуск

```bash
go run . --secret <ключ> [--mode cipher|decipher]
```

## Флаги

| Флаг | Описание | По умолчанию |
|------|----------|--------------|
| `--secret` | Секретный ключ для шифрования/дешифрования | обязательный |
| `--mode` | Режим работы: `cipher` или `decipher` | `cipher` |

## Примеры

**Шифрование:**
```
$ go run . --secret mykey
Enter text for cipher:
> hello world
GhIcGREAHgwWGw==
```

**Дешифрование:**
```
$ go run . --secret mykey --mode decipher
Enter text for decipher:
> GhIcGREAHgwWGw==
hello world
```

## Важно

- Для дешифрования нужно использовать **тот же ключ** что и при шифровании
- При копировании зашифрованной строки копируй её **полностью**, иначе Base64 декодирование завершится ошибкой
